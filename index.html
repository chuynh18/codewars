Arbitrary precision division
Enter an arbitrarily precise dividend and divisor and see it get divided! If you don't understand why this is a big deal, JavaScript does not natively support arbitrary precision division. (All numbers in JavaScript are doubles.)
This means that 0.3/0.2 === 1.4999999999999998. On the other hand, my code returns 0.3/0.2 === 1.5. Don't believe me? Open your browser's console (likely the F12 key), click on the Console tab, and try computing 0.3/0.2 for yourself!
Note that my code truncates rather than rounds!

Evaluate mathematical expression
Enter an arithmetic expression, meaning an expression containing any of the following operations: addition (+), subtraction (-), multiplication (*), and division (/). Expressions are evaluated left-to-right. Order of operations is respected including nested parentheses. Operations are evaluated using JavaScript's built in arithmetic operators (so you get double precision, NOT arbitrary precision).
Note: I will not parse your mathematical expression for correctness. If you don't get the result you expect, check your expression for typos.
Example: 45+(23-4)*(5+(5345-331))+333 === 95739
